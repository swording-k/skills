# 教师现场演示指南

为学生生成一份能够直接照着操作的现场演示说明。

## 每个实验必须包含

### 1. 启动方式

- 双击哪个文件，或执行什么命令；
- 从哪个目录启动；
- 需要先打开哪个软件、工程、数据或设备；
- 需要确认哪些依赖和权限。

### 2. 稳定演示输入

用表格列出每个输入框、参数、文件或设置应该填什么。选择能够快速产生清晰结果的数据。

### 3. 逐步操作顺序

每一步写清：

- 在哪里操作；
- 点击什么、输入什么或连接什么；
- 操作后应出现什么；
- 应向老师说明什么原理；
- 哪个结果证明满足指导书要求。

### 4. 真实性证明

至少准备一种方式证明结果不是固定输出或伪造：

- 改变一个输入，展示输出随之变化；
- 使用异常输入，展示验证或错误处理；
- 改变一个参数，展示曲线、测量值或结果变化；
- 重新运行实验，展示可复现结果；
- 打开原始工程或数据，展示结果来源。

### 5. 老师可能提问

准备简短回答：

- 这个实验的目的和原理是什么？
- 这一结果是如何得到的？
- 为什么选择这些参数或数据？
- 如果输入或参数改变，会发生什么？
- 遇到了什么问题，如何解决？
- 如何证明截图和结果是真实操作得到的？
- 实验有哪些限制和改进方向？

### 6. 现场故障恢复

说明：

- 软件打不开时如何重新启动；
- 工程或数据文件在哪里；
- 如何恢复默认参数；
- 如何快速重新运行关键步骤；
- 哪些步骤需要用户亲自完成。
